1. For WordPress beginners

Although building a website with WordPress is significantly easier than coding it by yourself, if you are a beginner, even working with WordPress may look hard for you. That's why, in this section we will provide some useful links and also will explain how to get oriented in the documentation and what pattern to follow when building your website with our theme.

Working with WordPress - Useful Links

  • WP101 Basic Training - the perfect beginning spot for those just getting started with WordPress. The video tutorials provided are very helpful to easily understand how WordPress works.
  • WordPress Codex - the official WordPress manual containing lots of information and tutorials about working with WordPress.

Installing the Theme

There are 2 ways to install WordPress theme.

Via builtin WordPress theme uploader

  • Go to your WordPress Dashboard and choose Appearance Themes from the left pane.
  • When Theme page arrived, click on Add New button.
  • Now in the Install Themes page click on Upload link which is just above of the Search box
  • Click on Choose File button and choose thefirstblog.zip file. Then click Install Now button.
  • After a few moments after installing the theme, you may be ready to enjoy your theme.

Via FTP Client

  • Install and run any FTP client. For example, you may use FileZilla FTP client. Its free and opensource and also available on Mac, Linux and Windows platform. We are writing this tutorial for FileZilla now. Although any other FTP client should be fine and their procedures are similar as FileZilla.
  • After running your FTP client software, give your FTP Host Name, Username, Password and Port Number (usually 21) on the respective field and click on Quick Connect.
  • After establishing connection to the server; from the right side of the folder list, visit WordPress directory where wp-admin, wp-content and wp-includes directories are situated.
  • Go to wp-content themes directory.
  • Now, minimize your FileZilla software and go to your thefirstblog.zip file location in your computer. Now unzip (or decompress or extract) that thefirstblog.zip file.
  • Go back to your FileZilla software and at the left side there is also a folder list almost similar to the right one. From that directory list locate your thefirstblog directory that you had decompressed few seconds ago.
  • After locating that directory from FileZilla, right click on thefirstblog directory and choose Upload.
  • Now, all of your files should be transferring from your PC to your web server via FTP client.
  • Completing the transfer, open your browser and go to your WordPress Dashboard.
  • Choose Appearance Themes.
  • When Themes page arrived, there should be a The First Blog theme on the list.
  • Choose your theme by clicking on Activate button.
  • Now, go back to your website and enjoy.

3. Gettings Started

After successfully installing the theme, it is time to start some tweaking. In this section we are discussing some fundamental settings specifically required for this theme at the beginning. Don't worry it won't take long.

Adding Menu

  • Go to Dashboard and from the left pane choose Appearance Menus.
  • After organizing or editing your menu, give it a name. The default name is Menu 1 on Menu Name field. You can change this to any name like My Menu and click on Create Menu or Save Menu button.
  • Now you menu is created and we named our menu as My Menu in this example.
  • Click Appearance Customize.
  • In the Customize page click on Navigation from the list in the left and select the menu you have created few seconds ago. In our example we are choosing My Menu from the drop down list.
  • Click on Save button.
  • Congratulation! Now visit your WordPress site to enjoy.

Getting Familiar with Theme Settings Page

If you are logged on your WordPress site, at the top of the page there is an admin bar. There you should find a menu item Theme Settings. Or in the dashboard, at the bottom of the left pane, there is also a menu item named Theme Settings. Click on that link to go to Theme Settings page.

In the, Theme Settings page you can change almost anything, like hiding, showing any element, changing any kind of fonts or changing any color of the element in depth. We will go through about this area one by one on later chapters.

Installing Required Plugins

Although this theme includes all the sufficient features, but there are 2 more plugins required to make this theme more awesome.

Slim Jetpack

Required for embedding shortcode. Install this plugin and then open Admin Panel Settings Slim Jetpack. Find Shortcode Embed option and click on Activate button.

If you are already using Jetpack by WordPress.com, then you don't need to use this Slim Jetpack plugin.

Contact Form 7
Although this theme already comes with "Contact Form", but also works well with Contact Form 7. Using Contact Form 7 is optional here but recommended.

4. Theme Settings Page

In this section we are going to discuss about some features of this theme, which are absolutely necessary to know for user to use the full potential of the theme.

(a) Adding Social Icons on Header

  • Go to Admin Panel Theme Settings Social Links
  • Enter your social links on respective fields.
  • Click on Save Changes button.

Make sure Disable social links on header check box is unchecked. Otherwise, you can't see any social links on header.

(b) Adding Tweeter Feeds on Header

You can display your Twitter feeds on website header. To achieve that, you need to have a Twitter Widget ID. Here are the required steps.

Aquire a Twitter Widget ID

  • Login in to Twitter account.

  • After logging in, go to Twitter Settings Page

  • Now choose Widget from the left pane.

  • When twitter widget page arrived, then click on Create New button.
  • Now, there is a Create a user widget page opened. Here you don't need to do anything. Just click on Create widget button.

  • Now at the bottom right side of the page, there is a text area with some HTML code. Select all those code and paste it to any text editor.

  • If you copy and paste those code into a text editor, then you may find almost similar code like we have here.

    <a class="twitter-timeline" href="https://twitter.com/GraphiteThemes" data-widget-id="453852961827868672">Tweets by @GraphiteThemes</a>
    
<script>!function(d,s,id){var js,fjs=d.getElementsByTagName(s)[0],p=/^http:/.test(d.location)?'http':'https';if(!d.getElementById(id)){js=d.createElement(s);js.id=id;js.src=p+"://platform.twitter.com/widgets.js";fjs.parentNode.insertBefore(js,fjs);}}(document,"script","twitter-wjs");</script>
  • Now, look carefully. At the first line after data-widget-id, there is a long number; something like 453852961827868672. Copy this number. Because, this is your twitter widget id.

Setting Twitter Widget ID on Theme Settings Page

  • Copy Twitter Widget ID. In our case, it is 453852961827868672. But you must have different number as your widget ID.
  • Go to Admin Panel Twitter Slider
  • Paste that Twitter Widget ID of yours and set the Total number of tweets
  • Click on Save Changes button

Make sure Disable Twitter slider from header check box is unchecked. Otherwise, you can't see any twitter feed on header.

(c) Changing Logo

Although this theme show site title in header, but you can change this to logo if you want.

  • Go to Admin Panel Theme Settings Header
  • At the custom logo field, either you can type eternal link of image of the logo or click on Upload button to upload your logo.
  • Alternatively you can also, use image from media library via Upload button to insert your custom logo.
  • Now click on Save button.

(d) Changing Favicon

  • Go to Admin Panel Theme Settings Header
  • At the favicon field, either you can type eternal link of image of the favicon or click on Upload button to upload your favicon.
  • Alternatively you can also, use image from media library via Upload button to insert your favicon.
  • Now click on Save button.

(e) Header Background

You can either change the background image or color of header

  • Go to Admin Panel Theme Settings Header
  • Choose Header background type
  • Now upload image or change color and click on Save Changes button

(f) Disable Header Animation

When you move your mouse over to any menu item, the header area animates. You can disable this feature.

  • Go to Admin Panel Theme Settings Header
  • Click on Disable header animation when mouse hovered on menu to check the checkbox
  • Now click on Save Changes

If you want to enlarge the header or if you are using very long list of sub menu along with Twitter Feeds on header, then it might be a good idea to enable the animation.

(g) Increase the Size of Header

  • Go to Admin Panel Theme Settings Header
  • Click on Disable header animation when mouse hovered on menu to check the checkbox
  • Now increase or decrease the number of Padding for header area
  • Now click on Save Changes

Make sure that Disable header animation when mouse hovered on menu is checked.

(h) Menu Opacity

You can also change the Transparency of menu from Admin Panel Theme Settings Header

(i) Author's Description

On every blog page, you can display author's description. WordPress already has that builtin feature.

  • Go to Admin Panel Users Your Profile

  • Now write about yourself on Biographical Info field
  • Click on Update Profile

(j) Author's Social Links

(k) Share Article

5. Useful Instructions

Setting up Featured Image

In the blog post or page, you can set up featured image. To do that:

  • Open blog/page editor.

  • Now at the lower right corner there is an option for Featured Image. Click on that link.

  • A media library will open, upload or choose image from media library for image featuring on your post.

Link Post

This theme can allow you to use link type of post. Your article title can directly link to external page instead of WordPress permalink. For example, on normal occasion, when you publish a blog post; its title links to a single article page inside of your WordPress site. But if you want to point your article title link to another site like http://www.facebook.com then you want to follow these steps:

  • Open Admin Bar New Post
  • Now at the top right corner of the page, just below to the Admin Bar there is a button Screen Options.
  • Click that Screen Options to expand and choose Custom Fields from there.
  • Now at the bottom of the page a new field called Custom Fields will be visible.
  • Expand that and click on Add Custom Field

  • Now type title_url and give your desired URL to its Value field.

  • Now at the right panel, choose Link from Format option.
  • Click on Publish button.

Creating a Gallery

  1. Go to Admin Bar New Post.

  2. Click on Add Media

  3. A Media Library window will open. Choose Create Gallery.
  4. Now choose images that you want to put on gallery.

  5. Click on Create New Gallery.

  6. Now on the Edit Gallery window, choose Link To Media File and Type to Thumbnail Grid from Gallery Settings at the right pane.

  7. Now, click on Insert Gallery button.

Creating a Gallery Slideshow

  • Install Slim Jetpack or Jetpack by WordPress.com plugin.
  • Follow the Creating a Gallery procedure from 1 to 5.

  • Now on the Edit Gallery window, choose Link To Media File and Type to Slideshow from Gallery Settings at the right pane.

  • Now, click on Insert Gallery button.

Portfolio Page

  • Go to Admin Bar New Portfolio.
  • On the new portfolio post, at the right side there are Project Options. Enter values for Skill, Client, Link etc.
  • You can also give Category for your project.
  • Set featured Image.
  • Click on Publish.

Portfolio Gallery

The First Blog theme can show your portfolio in a gallery. To accomplish that, you just need to create a page and set the template to Portfolio. Then it will automatically grab all of your projects/works and show them as gallery.

Contact Form Page

There are 2 different ways to create a contact form page.

Contact Form 7
  • Install and activate Contact Form 7 plugin.
  • Go to Admin Panel Contact Contact Form.

  • Copy the shortcode for Contact Form. This shortcode might be similar to this one.

    [contact-form-7 id="1570" title="Contact form 1"]
  • Create a new page and give it Default Template as Template.
  • Paste that shortcode that you copied before.
  • Click on Publish button.
Contact Form Page Template
  • Go to Admin Panel New Page.
  • Choose page Template as Contact.
  • Click on Publish button.

Archive Page

  • Go to Admin Panel New Page.
  • Choose page Template as Archive.
  • Click on Publish button.

7. Translation

This theme is translatable. We have included thefirstblog.pot file at thefirstblog (theme directory) languages folder. To open that file you may use Poedit software. This software is free and opensource and also available on Mac, Linux and Windows platform. So, download, install and run the Poedit software and follow procedures written in bellow.

  • Run Poedit software and open thefirstblog.pot file from languages directory.

  • Go to Catalog Properties...
  • A Catalog Properties window will open. Change Project Name, Version, Languages etc. Then press Ok button.
  • Now translate those English words to your language.
  • Click File Save as... from menu.
  • Well, the save part is little bit tricky. You will need to name as your-language-code_YOUR-COUNTRY-CODE. For example, if you are Spanish, the the file name should be es_ES.po or if you are French then it should be fr_FR.po
  • A list of country and language code can be found here.
  • Now, that you have saved your *.po and *.mo (this will be automatically created) file, its time to add your language to wp-config.php file.
  • Go to WordPress root directory where wp-admin, wp-content, wp-includes folder are located. There you will find wp-config.php file.
  • Now, open wp-config.php with any plain text editor (like Textmate, gEdit, Notepad etc).
  • In that file find define ('WPLANG', ''); and change this to that file name you saved (without extension). For example if your file name is fr_FR.po, then write define ('WPLANG', 'fr_FR');
  • Save the file and refresh your WordPress site.

  • HandleTwitterPost.js - For fetching Twitter feeds.

  • jquery.fitvids.js - For responsive embedding video. Downloaded from http://fitvidsjs.com/
  • jquery.flexslider-min.js - Slider used fro Twitter Feeds on header. Downloaded from http://www.woothemes.com/flexslider/
  • jquery.scrollUp.min.js - Downloaded from http://markgoodyear.com/labs/scrollup/

  • tfb-insert-shortcodes.js - This is the Shortcode Wizard dialog, meant for choosing and using shortcode graphically instead of writing codes on hand.
  • tfb-menu-script.js - Required for running responsive menu.
  • tfb-shortcodes-behaviour.js - This contains Javascript functions for running shortcode perfectly
  • TwitterPostFetcher.js - Contains core library for fetching Twitter feeds from internet
